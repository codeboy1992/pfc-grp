Aide - Comment jouer ?

Bienvenue dans le jeu Pierre, Feuille, Ciseaux ! Voici comment jouer :

  1. Entrez votre prénom dans le champ prévu.
  2. Cliquez sur le bouton Commencer.
  3. Choisissez entre Pierre, Feuille, ou Ciseaux.
  4. Attendez le décompte pour voir le choix de l'IA et le résultat de la manche.
  5. Le premier joueur à atteindre 3 points remporte la partie.
  6. Utilisez le bouton Recommencer pour jouer une nouvelle partie.

Bonne chance et amusez-vous bien !
